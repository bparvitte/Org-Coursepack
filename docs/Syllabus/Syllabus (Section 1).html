Course Description¶

This course provides students with Org-Coursepack, a modular and reusable teaching materials template in org-mode. Using self-explanatory course contents, students will be able to adapt the template and create their on course contents through Org-mode.

Course Prerequisites¶

  • Basic knowledge about using Emacs
  • Basic knowledge about org-mode

Student Learning Objectives¶

As the result of this course, students should be able to:

  • Adapt Org-Teaching-Template to create their own course contents

Course Material¶

Documentation
Handouts, readings, and assignments will be uploaded to the repository.

Software

  • Emacs. (Orgmode is a built-in package of Emacs)
  • org-reveal
